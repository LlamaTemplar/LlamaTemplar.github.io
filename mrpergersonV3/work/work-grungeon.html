Description

Grungeon was one of the 1,040 submissions for itch.io's biggest game jam of 2018, Game Maker's Toolkit Jam 2018

This game was created by me and my friend Thomas using Unity3d. The game consistent of procedural generated levels and three different enemies that were programmed by Thomas. All the visuals, animations, and UI were all created by me. In order to win the game, you had to traverse up the dungeon to find the key to unlock the escape door while being attacked by goblins.

Grungeon is now on display at the KALEID gallery in downtown San Jose.
